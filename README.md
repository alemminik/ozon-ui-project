## Требования и входные данные

- Java 17 или новее;
- Maven 3.9 или новее;
- Google Chrome;
- авторизованный профиль Ozon в `chrome-profile/`.

Исходные данные, указанные в `checklist.pdf`, пользователь подготавливает вручную перед запуском. Тесты выполняют только действия из чек-листа и не создают недостающие входные данные.

## Запуск

Все команды выполняются из каталога `ozon-ui-project`.

```bash
mvn test
```

Запуск одного класса:

```bash
mvn -Dtest=SvincovFavoritesTest test
```

Запуск одного метода:

```bash
mvn -Dtest=DenisenkovFavoritesTest#removeFromFavorites test
```

- `SvincovFavoritesTest` — тесты 1, 5, 10;
- `PobegajlovFavoritesTest` — тесты 2, 6, 9;
- `KarpovFavoritesTest` — тесты 3, 7;
- `DenisenkovFavoritesTest` — тесты 4, 8.

## Локальные файлы

Каталоги `target/`, `build/` и `chrome-profile/` не должны попадать в Git. В них находятся результаты сборки, отчёты Selenide, скриншоты и локальный профиль браузера.
