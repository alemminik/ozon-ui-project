# UI-тесты раздела «Избранное» Ozon

Проект содержит 10 UI-тестов на Java, JUnit 5 и Selenide по актуальной версии чек-листа.

## Состав тестов

| № | Проверка |
|---:|---|
| 1 | Корректное отображение раздела «Избранное» |
| 2 | Добавление в избранное со страницы товара |
| 3 | Добавление в избранное из результатов поиска |
| 4 | Удаление товара из избранного |
| 5 | Сохранение списка избранного после обновления |
| 6 | Обратный порядок добавления товаров |
| 7 | Фильтрация избранного по категории |
| 8 | Добавление товара из избранного в корзину |
| 9 | Синхронизация сердца в поиске и на странице товара |
| 10 | Подгрузка рекомендаций под разделом «Избранное» |

## Требования и входные данные

- Java 17 или новее;
- Apache Maven 3.9 или новее;
- Google Chrome;
- отдельный тестовый аккаунт, авторизованный в `chrome-profile/`.

Перед каждым UI-тестом проект через интерфейс Ozon полностью очищает избранное и корзину, затем создаёт необходимые входные данные. После теста созданные данные удаляются. Не используйте профиль одновременно в нескольких процессах: тесты настроены на последовательное выполнение и могут полностью очистить аккаунт.

## Запуск

Все команды выполняются из каталога `ozon-ui-project`.

```bash
mvn test
```

Запуск одного класса:

```bash
mvn -Dtest=FavoritesPageTest test
```

Запуск одного метода:

```bash
mvn '-Dtest=FavoritesProductManagementTest#shouldRemoveProductFromFavorites' test
```

- `FavoritesPageTest` — тесты 1, 5, 7, 10;
- `FavoritesProductManagementTest` — тесты 2, 3, 4, 6, 9;
- `FavoritesCartTest` — тест 8.

## Локальные файлы

Каталоги `target/`, `build/` и `chrome-profile/` не должны попадать в Git. В них находятся результаты сборки, отчёты Selenide, скриншоты и локальный профиль браузера.
